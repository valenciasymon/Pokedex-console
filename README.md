# Pokédex Console App

A console-based CRUD application built with C# and MySQL for managing Pokémon data.

## Features

* View all Pokémon
* Register Pokémon
* Search Pokémon
* Favorite Pokémon
* Delete Pokémon
* MySQL database integration

## Technologies Used

* C#
* .NET
* MySQL
* MySql.Data

## Database

This project uses a MySQL database named:

```sql
pokedex
```

Main table:

```sql
pokemons
```

## Sample Columns

* id
* pokemon_name
* element
* rarity
* isfavorite
